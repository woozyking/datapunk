Datapunk Performance Dashboard

Real-world analytics benchmarks across five data engines (with memory-aware twins) on NYC taxi data. Each suite runs twice: a single month uncapped (apples-to-apples), then the full 24-month window under a physical RAM cap that surfaces out-of-core behaviour. Engines that must materialise everything OOM; engines configured with memory limits spill and survive. ★ = best in run · Stress = peak RAM ÷ cap (lower is better).
